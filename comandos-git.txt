 git init - iniciar git en repositorio

git status - conocer el estado actual de mi repositorio git 

git ignore - ignorar carpetas o archivos que no necesitemos que sean subidas

git add . - con este le indicamos a git que guarde todos los archivos nuevos que detecte

git commit -m "mensaje indicativo del cambio" - con este comando confirmamos todos los cambios que hicimos hasta ese momento.

git log - genera un historial completo de los ultimos commits, usuario, fecha y hora de esos cambios

git log --oneline - genera un historial resumido 

git merge nombrerama - fusiona los cambios de una rama dentro de la rama principal.

                        tener en cuenta que para hacer el merge, la rama no debe tener add ni commit -m name
